MisinfoX is a privacy-first Chrome browser extension designed to help users detect potential misinformation, scams, and misleading content in real time. Users can select any text on a webpage and instantly analyze its credibility without navigating away from the page.

Overview
MisinfoX enables real-time verification of online content directly within the browser. By analyzing selected text using a heuristic and AI-assisted backend system, the extension provides users with clear credibility assessments to promote informed decision-making while browsing.

Key Features
-Text-based verification through user selection
-Real-time detection of false, misleading, or suspicious content
-Context-aware analysis considering credibility and source reliability
-Privacy-first design with no storage of browsing history or personal data
-Seamless in-page verification without redirecting users

Technology Stack
-Frontend: HTML, CSS, JavaScript
-Styling: CSS
-Browser Platform: Chrome Extensions API
-Icons: Feather Icons
